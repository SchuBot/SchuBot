> The concept is that SchuBot allows you to have all features available on a single page without having to navigate to other pages to get access to what you need.

# SchuBot



SchuBot is currently in Beta v2.0.0 (some features currently under development)

Beta Testing now underway.

Authentication - Authenticate with your Streamer and Bot account

UI Theme ‚Äì Allows you to change the appearance of UI controls such as buttons, module panels , drop-down controls etc...

Chat Module Panel ‚Äì Allows you to see and reply to Mixer/Twitch messages/whispers (More functions coming soon)

Viewer List ‚Äì To see who is on chat

Whisper Window ‚Äì To see your whispers (under development - Due July 2020)

Recent Followers List ‚Äì Shows your recent (last 24hrs) followers. (under development Due July 2020 )

Following List ‚Äì Shows a list of the people you follow.

Follower List ‚Äì Shows your follower list

Analytics (under development)

Commands List ‚Äì Allows you to create commands that can be triggered in chat.

Media Files ‚Äì To setup your files for alerts

Alerts ‚Äì The setup the actual alerts.

Keywords ‚Äì These are words that trigger a response from a chat message

Notes  ‚Äì Allows you to create reminders and/or to be used as a to-do list

Moderation Monitor ‚Äì Allows you to see what mods have been up to.

Timers ‚Äì Messages sent to chat on specific intervals

Currency and Rank - This is close to finished and will be released in V2.1.0 (Due July 2020)

and More features to come.
